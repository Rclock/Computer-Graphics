Detailed Description

Contains vertex and normal data for the Utah teapot.

Usage

Using this class is simple, first declare a teapot object, call the load method where the parameters are the data positions of the vertices and the normals expected by the shader, and finally call the draw method in the display function of your program.

teapot pot;
...
pot.load(0, 2);
...
pot.draw();

Data Format

VBO in the format tx,ty,nx,ny,nz,vx,vy,vz Data for a unit teapot stored in GL_T2_N3_V3 format
