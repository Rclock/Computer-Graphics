◆ display()

void GraphicsEngine::display ( )

The function responsible for drawing to the OpenGL frame buffer.

The display function draws all of the planets, the spaceship in it's current orientation, all of the lights, performs all transformations, plays sounds, and puts on proper textures. It also takes care of updating all of these things. It also prints all of the freetype.
