Detailed Description

Axes for development, allows the programmer to see the positioning of objects easier.

Draws the xyz axes on the screen, each axis has the coordinate range of [-1, 1]. The x-axis is in red, with the positive x-axis is bright red and the negative x-axis in a darker red. The same is true for the y-axis in green and the z-axis in blue.
