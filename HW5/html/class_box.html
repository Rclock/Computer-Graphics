Detailed Description

The Box class holds position and color data for a multicolored rectangle. It also has facilities for loading the data to the graphics card and invoking draw commands on the data.

GLfloat Box::getWidth ( )

Changes the color of the selected vertex.

Parameters
v— Number of the vertex to alter. Valid numbers are 0, 1, 2, 3, with 0 representing the upper left vertex and then moving clockwise for the other three vertices.
r— Red intensity for the vertex.
g— Green intensity for the vertex.
b— Blue intensity for the vertex.

Resets the color of vertex v to the new (r, g, b) value and calls the data loading method. Returns the width of the rectangle.
